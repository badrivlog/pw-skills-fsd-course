HTML
HTML stands for Hyper Text Markup Language, is used for creating the structure and content of web pages including heading, paragraph, forms, tables, links, images and more.
CSS
CSS stands for Cascading Style Sheet, it allows how the web content look like, means adding style to the web content. it controls beatification part of the webpage. By CSS can control layout, colors, fonts, everything shownig on the web page.
JavaScript
A programing language that adds interactivity to web pages. currently JavaScript is the most popular programing language commonly used for web development. if a button clicked show a dialog box or submit a form, that kind of interactivity done by JavaScript
React.js
A javascript library for building user interfaces maintaining by Facebook. ReactJs is all about components, it makes easy to create modern and complex user interfaces. it helps developers create dynamic, interactive, and efficient UI by breaking down the interface into reusable components
Node.js
Node.js is a free and open-source runtime environment that allows to run JavaScript on the server side, outside of a browser. It is built on Chrome's V8 JavaScript engine,
Express.js
A lightweight and fast web framework for nodejs to build server-side application and APIs. Express.js is used by PayPal, Uber and IBM
MongoDB
MongoDB is a popular NoSQL database that is used for storing and managing data in a flexible, scalable way. unlike traditional relational databases that use tables to store data, MongoDB uses a document-oriented approach where data is stored in BSON format

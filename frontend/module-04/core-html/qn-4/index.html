1. <html> :The root elememt that wraps all the content of html document.
  2. <head>: Contains metadata about the html document, like the title, links to stylesheets, and meta information.
  3. <title>: Specifies the title of the html document, which appears in the browser tab or title bar.
  4. <body>: Contains the visible content of the html document, such as text, images, and other elements.
  5. <div>: Creates division or section in html.
  6. <h1> to <h6>: For titles or subtitles.
  7. <p>: Represents a paragraph of text.
  8. <a>: Defines a hyperlink, which is used to link one page to another or to specific parts of the same page.
  9. <img>: Embeds image in the html document.
  10. <span>: An inline container element used for styling a small portion of text or other inline content.
  11. <ul>: Defines unordered list.
  12. <ol>: Defines ordered list.
  13. <li>: Defines a list item within either an unordered or ordered list.
  14. <table>: Represents a table in html document.
  15. <tr>: Defines a table row.
  16. <td>: Defines a cell in a table row.
  17. <th>: Defines a header cell in a table.
  18. <form>: Defines an html form for collect user data.
  19. <input>: Represents an input field in a form, such as a text box, checkbox, or button.
  20. <button>: Defines a clickable button.
  21. <storng>: Represents important text, rendered bold text.
  22. <em>: Represents emphasized text, rendered in italics.
  23. <br>: Creates a line break within a block of text.

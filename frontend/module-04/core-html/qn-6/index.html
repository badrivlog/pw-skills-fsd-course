1. HTML

    Primary use cases Key features/Benfits
    Building structure of web pages
    • simple and easy to learn
    • compatible with all web browsers
    • allows for semantic markup

  2. CSS

    Primary use cases Key features/Benfits
    Styling and layout of web pages
    • allows for separation of content and presentation
    • enables responsive design
    • offers a wide range of styling options

  3. Javascript

    Primary use cases Key features/Benfits
    Adding interactivity and functionality to web pages
    • supports dynamic content updates
    • works on both client-side and server-side
    • compatible with all modern web browsers

  4. React.js

    Primary use cases Key features/Benfits
    JavaScript library for building user interfaces
    • component-based architecture
    • virtual DOM supports
    • rich eco-system

  5. Node.js

    Primary use cases Key features/Benfits
    JavaScript runtime for backend development
    • non-blocking I/O
    • asynchronous execution
    • event-driven architecture

  6. Express.js

    Primary use cases Key features/Benfits
    Web application framework for Node.js
    • simplified routing
    • middleware support
    • easy to use with Node.js

  7. MongoDB

    Primary use cases Key features/Benfits
    NoSQL, document-based database
    • flexible schema
    • horizontal scaling
    • high performance
